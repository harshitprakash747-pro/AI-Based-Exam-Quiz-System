# AI-Based Exam Quiz System

## About the Project
This is a simple quiz application written in C.
The program allows the user to choose a subject and attempts
important exam-related questions based on priority.

## Features
- Subject selection (Computer Science, Physics, Maths)
- Important questions filtering using priority level
- Multiple-choice questions
- Score calculation
- Performance feedback at the end

## Technologies Used
- C Programming
- VS Code
- GitHub
- ChatGPT (to identify and correct mistakes in code)

## How to Run the Program
1. Open the source file in any C compiler
2. Compile the program
3. Run the executable
4. Answer the questions shown on the screen

## Note
The questions included are basic sample questions.
More questions and features can be added in future versions.
